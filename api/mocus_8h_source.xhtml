18 /// @file
19 /// Fault tree analysis with the MOCUS algorithm.
20 /// This algorithm requires a fault tree in negation normal form.
21 /// The fault tree must only contain AND and OR gates.
22 /// All gates must be positive.
23 /// That is, negations must be pushed down to leaves, basic events.
24 /// The fault tree should not contain constants or house events.
