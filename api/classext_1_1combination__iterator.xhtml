Detailed Description

template<class Iterator, class value_type = std::vector<typename Iterator::value_type>>
class ext::combination_iterator< Iterator, value_type >

Read forward-iterator for combination generation.

The combination generator guarantees the element order to be the same as in the original source collection.

Template Parameters
