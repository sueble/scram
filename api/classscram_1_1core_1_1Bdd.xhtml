Detailed Description

Analysis of PDAGs with Binary Decision Diagrams.

This binary decision diagram data structure represents Reduced Ordered BDD with attributed edges.

Note
The low/else edge is chosen to have the attribute for an ITE vertex. There is only one terminal vertex of value 1/True.
