fuzz_tester module¶

Runs SCRAM with various input files and configurations.

This script is helpful to detect rare bugs, failed assumptions, flawed design, and bottlenecks. For the full functionality of the Fuzz tester, SCRAM must be built in non-Release mode so that NDEBUG is undefined.

In addition to its main purpose, the script is handy to discover complex auto-generated analysis inputs and configurations.
