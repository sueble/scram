Detailed Description

The Dialog to create, present, and manipulate event data.

Only valid data is accepted by this dialog. That is, the dialog constrains the user input to be valid, and upon the acceptance, it guarantees that the data is valid for usage by the Model classes.

However, the requested data must be relevant to the current type.

Precondition
The model is normalized.
